# 🎁 Happy Moments – Ultra-Colorful Gifting Website

## 📌 Overview

Happy Moments is a modern, vibrant e-commerce frontend project designed to showcase gift items with an engaging UI and interactive features.

## ✨ Features

* 🎨 Colorful modern UI with gradients
* 🛍️ Product filtering system
* 🛒 Shopping cart with localStorage
* 🔍 Live search functionality
* 🌙 Dark/Light theme toggle
* 📱 Responsive design

## 🧰 Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```
├── index.html
├── style.css
├── script.js
```

## 🚀 How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start exploring the website

## 🛒 How It Works

* Products are loaded dynamically
* Add items to cart using "+" button
* Cart is saved in browser storage
* Search filters products instantly
